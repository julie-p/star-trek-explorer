# Star Trek Explorer App

A Star Trek Angular App using the [Star Trek API](http://stapi.co/) to retrieve episodes of the show.
Project made in 6h as a technical interview for [Flitdesk](https://www.flitdesk.com/fr/).

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 11.0.5.

## Development server

Clone the app and run `npm install`. Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
